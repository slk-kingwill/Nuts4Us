## Prerequisites
```
Before running the Nuts4Us Website, you need to have the following set up:
  - Node
  - PostgreSQL (>= 9.5)
  - A .env file with the DATABASE_URL environment variable defined (not included in this repository for safety reasons).
```

## Requirements
```
cd backend/
npm install
```

## Steps to run
```
cd backend/
node app.js
```
